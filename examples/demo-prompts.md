# Example Agentic AI Prompts

These prompts demonstrate agentic AI capabilities — going beyond simple Q&A into multi-step autonomous task execution.

---

## 🏗️ Software Architecture (Impressive for Demos)

```
Act as a senior software architect.
Create a complete specification for a Room Booking System for HELP International School.

Tasks:
1. Gather functional requirements
2. Create user roles
3. Create SQL database schema
4. Create system architecture
5. Create folder structure
6. Create API endpoints
7. Create UI page list
8. Create development roadmap

Output as a professional software requirements document.
```

## 🚀 Why These Prompts Are "Agentic"

Normal chatbot: Ask question → Get answer → Done.

Agentic AI:
1. Breaks the task into subtasks automatically
2. Creates a plan
3. Writes files / generates code
4. Reviews its own output
5. Improves and iterates
6. Produces a complete deliverable

**One prompt → complete project blueprint.**  
That's the difference between a chatbot and an AI agent.
